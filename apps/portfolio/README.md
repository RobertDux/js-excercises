# Portfolio

## Vereisten

- De broncode bestaat alleen uit HTML5 en CSS3
- Er mag geen gebruik worden gemaakt van frameworks
- Het portfolio bestaat uit een enkele webpagina
- De pagina bevat een overzicht van je afgeronde "opdrachten"
- De pagina bevat een overzicht van je vaardigheden
- De pagina bevat een stukje over "jezelf"
- De pagina bevat reacties van "tevreden klanten"
- De pagina bevat een contactformulier met ten minste drie velden
- Het portfolio bevat ten minste drie afbeeldingen
- De pagina is responsief (i.e. werkt goed op ieder device)
